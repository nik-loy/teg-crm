**Welcome to the new TEG CRM!** 👋

This is your daily tool for LinkedIn outreach. Everything you used to do in the CLI (contact_logger, message_gen) is now here in one web app.

### 3. You're In!

## 📋 What You'll See

### **Today Tab** (Your Daily Dashboard)
Shows all the contacts you need to reach out to:
- **"Request Sent" (no response yet)** — Follow-up candidates
- **"Stale" contacts** (no message for 5+ days) — Needs check-in
- One-click to view contact or generate message

### **Contacts Tab** (Full Table)
All contacts, sortable by:
- Pipeline Stage (Awareness → Messaged → Engaged → Deepening)
- Owner (you, Finn, etc.)
- Date added
- Company
- Status

Search by name, company, or email. Click a contact to see full details.

### **Pipeline Tab** (Kanban Board)
Visual view of the sales funnel:
- **Awareness** → just connected
- **Messaged** → sent first message
- **Engaged** → they replied positively
- **Deepening** → follow-up conversations
- **Activated** → ready for next step

Drag cards between columns to update status.

### **Messages Tab** (Message Generator)
Your main outreach tool:
1. **Select a contact** (or create new)
2. **Paste LinkedIn profile** (headline, bio, experience, posts) — or click bookmarklet button
3. **AI generates 3 message variants** on different angles
4. **You pick the best one**, edit if needed, copy to clipboard
5. **Log to Notion** and send manually on LinkedIn

### **Add Contact Tab**
Two ways to add:

**Option A: Use the ＋TEG Bookmarklet** (Recommended)
1. Click "Install Bookmarklet" button
2. Drag the blue "＋TEG" button to your browser bookmarks
3. From now on: Go to LinkedIn profile → Click ＋TEG bookmarklet
4. App pre-fills name + LinkedIn URL → Click Add

**Option B: Manual Entry**
Click "Add Manually" → Fill in name + LinkedIn URL → Paste full profile text (optional) → Click Add

### **Screenshots Tab** (Batch Processing)
If you captured LinkedIn screenshot cards:
1. Upload screenshot
2. AI extracts names + companies automatically
3. System deduplicates and creates contacts
4. (Optional — most teams just use the bookmarklet)

## 💬 Message Generation Explained

When you paste a LinkedIn profile, the AI generates **3 variants**:

### **Variant 1: "External, Du" (Casual)**
Best for: Young professionals, first-time outreach
```
Hey [Name], danke fürs Vernetzen! Hab gesehen, dass du viel mit 
[their focus] machst. Wir machen [event description]. Könnte gut 
passen. Falls spannend: [link] VG Finn
```

### **Variant 2: "Internal, Sie" (Formal)**
Best for: People from speaker companies, more senior folks
```
Hallo Herr/Frau [Name], danke für die Vernetzung! Da Sie bei 
[company] arbeiten: Wir machen [event]. Ein Partner aus Ihrem Haus 
ist Speaker. Könnte thematisch passen. Falls Sie reinschauen 
möchten: [link] Beste Grüße, Finn
```

### **Variant 3: "Re-engagement" (Angle 2)**
Best for: Different angle on same person, follow-up
```
Hi [Name], kurze Gedanke nach deinem Profil — dein Fokus auf 
[specific skill] passt super zu [event topic]. [Event details]. 
Nur für Leute wie dich! [link]
```

**You pick the best one, edit if needed, then copy & send manually on LinkedIn.**

## 🔐 Security & Best Practices

### ✅ DO:
- Copy messages from app → paste on LinkedIn manually
- Add contacts via bookmarklet (fastest, least error)
- Check "Stale" contacts daily (find unresponsive leads)
- Log all messages (so team knows who you've contacted)
- Use "Du" for peers, "Sie" for senior folks

### ❌ DON'T:
- Share the app password on Slack (ask in person)
- Click other team's contacts (respect ownership)
- Edit another person's notes without asking
- Send LinkedIn messages from the app (it doesn't support that — by design, ban-safe)
- Paste full profile including private info (headlines + public text only)

## 🆘 Troubleshooting

### "Wrong password"
→ Double-check you're typing exactly right (case-sensitive). Ask Finn if forgotten.

### "Contact not loading"
→ Refresh page. If still down, Notion might be slow. Wait 10 sec.

### "Message generation takes forever"
→ Normal (AI takes 3–5 sec). Let it finish. Don't close the tab.

### "Bookmarklet doesn't work"
→ Make sure you installed it correctly (see "Add Contact" section). Test on a real LinkedIn profile page.

### "Can't find contact in search"
→ Try searching by company name instead of full name. Search is fuzzy but needs at least part of a word.

### "App keeps signing me out"
→ Session is 12 hours. Log back in. (Or ask Finn to extend timeout if needed.)

## 📊 Understanding Pipeline Stages

| Stage | Meaning | Next Action |
|-------|---------|------------|
| **Awareness** | Connected, haven't messaged yet | Message them with event info |
| **Messaged** | Sent first outreach | Wait for reply (5–7 days) |
| **Engaged** | They replied positively | Follow up with next step |
| **Deepening** | Active conversation | Build relationship |
| **Activated** | Ready for event/speaker/sponsor | Invite to specific next step |

**Pro tip:** Your goal each week is to move as many from "Awareness" → "Messaged" as possible.

## 🎨 Tips & Tricks

### Faster Contact Addition
- Use the ＋TEG bookmarklet on LinkedIn profiles (pre-fills name + URL)
- Paste full profile for better message personalization
- System auto-fills missing fields where it can

### Better Messages
- Include **2–3 specific details** from their profile (job title, skills, posts)
- Reference the **specific event topic** they care about
- Keep it **under 500 characters** (app shows count)
- Use **Du for peers, Sie for managers+**

### Higher Response Rates
- Message within **2–3 days** of connection request (strike while hot)
- Follow up after **5 days** if no reply (with variant #2 or #3)
- Mention **specific speakers** if they're from a speaker company
- Keep **Friday messages brief** (people are less active)

### Team Coordination
- Check "Contacts" tab to see who messaged whom (avoid duplicates)
- Use "Owner" filter to see your own contacts
- Read team notes before following up (might have context)
