# TEG CRM Setup Guide

> This guide will be filled in by Claude Code as the system is built.
> It will contain step-by-step instructions for a non-developer club member
> to set up and use the CRM automation system.

## Prerequisites
- Python 3.11+
- A Notion account with the TEG workspace
- GitHub account (for scheduled automations)

## Setup Steps
1. Clone this repository
2. Create `.env` from `.env.example`
3. Set up Notion integration and databases
4. Run initial data import
5. Configure GitHub Actions for scheduled tasks

(Detailed instructions will be added as each component is built.)
